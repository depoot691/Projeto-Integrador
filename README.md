- RPG Packs E-commerce -

📌 Descrição do Projeto

O RPG Packs E-commerce é uma plataforma de compras online de pacotes de RPG narradas por IA, desenvolvida para proporcionar uma experiência gamificada aos usuários. O sistema inclui:

Barra de progresso gamificada para rastrear pedidos.
Integração com uma IA para tirar dúvidas sobre produtos e suporte.
Processamento de pagamento (simulado).

O projeto combina tecnologias modernas de frontend e backend para criar um ambiente interativo e seguro para os usuários.

🛠️ Tecnologias Utilizadas
- Frontend:
HTML, CSS e JavaScript
Bootstrap para componentes e responsividade

- Backend:
Java + Spring Boot
API REST para comunicação entre front e back
MySQL como banco de dados relacional

- Extras:
Sistema de processamento de pagamento (fictício)
API de IA para atendimento e dúvidas dos usuários
Docker para containerizar frontend, backend e banco de dados

⚡ Funcionalidades
- Catálogo de Packs de RPG: Lista de pacotes disponíveis para compra.
- Packs de jogos RPG narrados diretamente por uma inteligência artificial. 
- Gamificação no rastreamento: Barra de progresso mostrando etapas do pedido.
- Interação com IA: Usuário pode tirar dúvidas sobre produtos ou funcionalidades.
- Adicionar ao carrinho e finalizar compra: Simulação do pagamento.
- Gerenciamento de pedidos: Backend registra pedidos no banco de dados MySQL.

🌐 Como acessar

O sistema é acessível como um site, sem necessidade de instalação.

Abra o navegador de sua preferência.
Digite o endereço do site onde o sistema está hospedado (local ou online).
Explore as funcionalidades:
Adicionar tarefas/packs ao carrinho
Acompanhar progresso gamificado
Tirar dúvidas com a IA
Limpar lista ou finalizar compra (simulada)

💡 Observações
O sistema de pagamento ainda é fictício, não realiza transações reais.
A API de IA é utilizada para simulação de atendimento, podendo ser integrada futuramente com APIs reais.
O projeto está em constante desenvolvimento e será expandido com novas funcionalidades gamificadas.
